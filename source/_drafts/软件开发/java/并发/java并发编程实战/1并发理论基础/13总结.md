- 面向对象编写并发程序
```
logger.debug("The var1：{}, var2:{}", var1, var2);
logger.debug("The var1：" +  var1 + ", var2:" + var2);
```
占位符写法仅仅是将参数压栈，而没有参数的计算。使用{}占位符是写日志的一个良好习惯。
